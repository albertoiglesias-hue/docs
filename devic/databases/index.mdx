## Descripción general

El módulo de **Bases de datos** de Devic proporciona un entorno nativo para almacenar, consultar y gestionar información estructurada sin depender de servicios externos.  
Está diseñado específicamente para integrarse con **agentes** y **MCPs**, permitiendo que la inteligencia artificial acceda a los datos de forma directa y contextual.

Cada base de datos se comporta como un repositorio optimizado para flujos impulsados por IA: puede ser utilizada por los agentes en tiempo real, conectarse con herramientas personalizadas o servir como origen de conocimiento para asistentes.

---

## Características principales

- **Gestión unificada:** administra todas tus bases de datos desde un único panel.  
- **Escalabilidad ilimitada:** crea tantas bases como necesites sin configuración adicional.  
- **Flexibilidad total:** añade, modifica o elimina campos y registros de forma dinámica.  
- **Búsqueda semántica integrada:** todas las tablas incluyen **Vector Search**, lo que permite realizar consultas por significado en lugar de coincidencia exacta.  
- **Interoperabilidad completa:** las bases de datos pueden ser consumidas por **agentes** y **MCPs** sin necesidad de conectores externos.  

---

## Panel de vista general

La sección **Databases** muestra un resumen de todas las bases activas y su estado actual.

En la parte superior, se presentan cuatro indicadores clave:

- **Total Databases:** número total de bases creadas.  
- **Total Rows:** cantidad global de registros almacenados.  
- **Custom Fields:** total de campos personalizados definidos por el usuario.  
- **Vector Search:** indica cuántas bases tienen activada la búsqueda semántica.

Cada fila del listado incluye:

| Columna | Descripción |
|----------|-------------|
| **Database Name** | Nombre de la base de datos. |
| **Description** | Breve descripción o alias del conjunto de datos. |
| **Total Rows** | Cantidad de registros almacenados. |
| **Custom Fields** | Número de columnas personalizadas. |
| **Vector Search** | Estado del buscador semántico (Enabled/Disabled). |
| **Actions** | Acceso directo a la vista detallada de la base mediante el botón **View**. |

---

## Acciones disponibles

Desde la parte superior derecha del panel puedes:

- **Crear una nueva base de datos** con el botón **+ Create New Table**.  
- **Explorar una base existente** seleccionando **View** en la columna *Actions*.  

Estas acciones te llevan a las secciones dedicadas:  
[Crear nueva base de datos](/devic/databases/create-new-database) y [Ver y editar base de datos](/devic/databases/view-and-edit-database).
