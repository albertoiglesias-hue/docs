## 1. Seleccionar tu proveedor de modelo

Al iniciar sesión en Devic, el primer paso es seleccionar el proveedor de modelo que utilizarás para ejecutar tus agentes y asistentes.

En la parte superior de la interfaz encontrarás el selector de modelos, donde puedes elegir entre:

- **OpenAI**
- **DeepSeek**
- **Anthropic**
- **XAi**
- **Kimi-K2**

<Card
  title="Configuración de proveedor"
  icon="gear"
  horizontal
>
  Haz clic en el icono de configuración junto al nombre del proveedor para definir tu proyecto y clave de API.
</Card>

### Configuración del proyecto

En el cuadro de diálogo **OpenAI Project Configuration**, introduce los datos correspondientes:

- **Project Name:** nombre descriptivo del proyecto o entorno.  
- **Project API Key:** clave del modelo o proveedor (se guarda de forma segura).  

Esto permite asociar ejecuciones y costes a proyectos concretos dentro de tu organización.

## 2. Dashboard

El **Dashboard** centraliza toda la actividad de tus agentes y asistentes.  
Desde aquí puedes supervisar en tiempo real las ejecuciones, estados y métricas clave.
