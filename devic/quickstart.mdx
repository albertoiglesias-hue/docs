## 1. Seleccionar tu proveedor de modelo

Al iniciar sesión en Devic, el primer paso es seleccionar el **proveedor de modelo** con el que se ejecutarán tus agentes y asistentes.

En la barra superior encontrarás el selector de modelos, con soporte para los siguientes proveedores:

- **OpenAI**
- **DeepSeek**
- **Anthropic**
- **XAi**
- **Kimi-K2**

<Card title="Configuración de proveedor" icon="gear" horizontal>
  Haz clic en el icono de configuración junto al nombre del proveedor para autenticarte con tu clave de API.
</Card>

### Autenticación con el proveedor

Antes de poder crear o ejecutar agentes, debes autenticar tu cuenta con la **API Key del modelo seleccionado**.\
Este paso es obligatorio: la plataforma no permite avanzar hasta que la autenticación se complete correctamente.

En la ventana de configuración introduce:

- **Project Name:** nombre identificativo del proyecto o entorno.
- **Project API Key:** clave API proporcionada por el proveedor del modelo.

![Autenticación de proveedor](/APIKeymodel.png)

Para obtener tu clave API, consulta las guías oficiales:

- [Cómo generar una API Key en OpenAI](https://platform.openai.com/account/api-keys)
- [Cómo generar una API Key en Anthropic](https://docs.anthropic.com/en/api/getting-started)
- [Cómo generar una API Key en DeepSeek](https://platform.deepseek.com/docs)
- [Cómo generar una API Key en XAi](https://x.ai/api)
- [Cómo generar una API Key en Kimi-K2](https://kimi.moonshot.cn/docs)

Una vez autenticado, el modelo quedará disponible para su uso en agentes, asistentes y MCPs.
