# Built-in Tools

Las **Built-in Tools** son herramientas nativas incluidas en Devic que cubren las operaciones más habituales en entornos agénticos y de asistentes.  
Su objetivo es **reducir al máximo la necesidad de desarrollos adicionales** o integraciones externas, proporcionando funciones listas para usar en tareas de automatización, consulta, análisis, comunicación y gestión de datos.

Estas herramientas pueden ser invocadas desde **Agentes**, **Asistentes** o **MCPs**, y están disponibles dentro del selector de herramientas al configurar cualquiera de ellos.

---

## 1. Update Knowledge

La herramienta **Update Knowledge** permite mantener actualizada la base de conocimiento de los agentes y MCPs mediante operaciones de escritura o actualización.

- **Tool:** `update_memory`  
- **Descripción:** Actualiza la memoria RAG con el contenido proporcionado.  
- **Uso recomendado:**  
  - Incorporar nueva información a una base de conocimiento.  
  - Actualizar datos o resúmenes que los agentes usan en sus respuestas.  
  - Sincronizar conocimiento desde documentos o APIs externas.

---

## 2. Send Email

La herramienta **Send Email** permite enviar correos electrónicos de forma automatizada directamente desde Devic.

- **Tool:** `send_basic_email`  
- **Descripción:** Envía un correo electrónico a la dirección especificada y, opcionalmente, escucha una respuesta.  
- **Uso recomendado:**  
  - Notificaciones automáticas.  
  - Envío de reportes o resultados de análisis.  
  - Confirmaciones o alertas internas.

![Send Email Tool](64cea700-c6f5-4476-894c-dbbc894d84d5.png)

---

## 3. OCR

La herramienta **OCR** (Optical Character Recognition) extrae texto de imágenes o documentos escaneados y permite realizar consultas sobre su contenido.

- **Tools disponibles:**  
  - `ocr_document`: extrae texto desde una imagen o documento.  
  - `query_document`: permite hacer preguntas en lenguaje natural sobre un documento PDF.  

![OCR Tools](ac2c1adb-c162-4ef9-a51f-efef332d01da.png)

---

## 4. PDF from HTML

**PDF from HTML** genera documentos PDF a partir de contenido HTML renderizable.

- **Tool:** `generate_pdf_from_html`  
- **Descripción:** Convierte HTML en un PDF con formato visual.  
- **Uso recomendado:**  
  - Exportar vistas web o plantillas con estilo corporativo.  
  - Generar informes o documentos con estructura HTML.

![PDF from HTML Tools](0b354075-42a6-4be0-a00f-e76574187f6c.png)

## 5. Code Execution Tools

Permite ejecutar código en diferentes lenguajes directamente desde Devic.

- **Tool:** `execute_python_code`  
- **Descripción:** Ejecuta código Python y devuelve el resultado o error correspondiente.  
- **Uso recomendado:**  
  - Calcular resultados dinámicos.  
  - Procesamiento de datos con scripts personalizados.  
  - Pruebas automatizadas o validaciones de código.

![Code Execution Tools](b7445a5c-afc9-4155-bc0d-d518a28f34e2.png)

## 6. Web Search

**Web Search** permite realizar búsquedas de información actualizada en la web.

- **Tools disponibles:**  
  - `search_web`: busca información relevante en la web.  
  - `web_extract`: devuelve el contenido de una URL específica.  

- **Uso recomendado:**  
  - Consultas actualizadas en fuentes externas.  
  - Ampliar conocimiento sin depender de memoria local.

![Web Search Tools](5b85d9eb-07c8-4aa6-87fe-811dffa9ee97.png)

  - `create_multiple_tasks`

  - Automatización de creación y seguimiento de tareas.

  - `create_folder`

## 9. Chart Images

Genera gráficos visuales a partir de datos y series numéricas.

- **Tool:** `generate_chart_image`  
- **Descripción:** Crea gráficos de tipo barra, línea o área, generando una URL de imagen.  
- **Uso recomendado:**  
  - Visualización de resultados analíticos.  
  - Inclusión de gráficos en reportes o dashboards.

![Chart Images Tools](ba2618d7-4cff-4b40-9223-9393c955cf4b.png)

## 10. Search Knowledge

Permite a los agentes buscar información relevante dentro de bases de conocimiento.

- **Tool:** `retrieve_from_rag_in_file`  
- **Descripción:** Busca información relacionada con la consulta del usuario en documentos del modelo RAG.  
- **Uso recomendado:**  
  - Consultas contextuales a documentación o datasets.  
  - Recuperar conocimiento relevante durante conversaciones.

![Search Knowledge Tools](4c5e144c-3c1d-4f77-a991-4279d788f9b2.png)
