## Crear un nuevo MCP

Para iniciar la creación de tu propio MCP, accede a la sección **MCPs → Mis MCPs** en el menú lateral y haz clic en el botón **Add custom MCP**.

<Card title="Añadir un MCP personalizado" icon="plus" horizontal>
  Desde este botón podrás importar tu API existente o definir un nuevo MCP desde cero.
</Card>

![Vista del botón Add custom MCP](/add-custom-mcp.png)

---

## Seleccionar modo de importación

Una vez abierto el asistente de creación, se despliega el cuadro **Import Tool Server**, donde podrás elegir el método de importación.

Las opciones disponibles son:

- **OpenAPI URL:** importa directamente la especificación pública de tu API mediante su URL.
- **OpenAPI JSON:** carga un archivo local con la especificación de tu API.
- **Start from scratch:** crea un MCP vacío para definir manualmente tus tools y endpoints.

<Card title="Importar desde OpenAPI" icon="link">
  Si tu API ya tiene una definición OpenAPI, basta con pegar la URL o subir el archivo JSON.\
  Devic generará automáticamente las tools y endpoints correspondientes, levantando de forma inmediata el servidor MCP para que los agentes puedan consumirlo.
</Card>

![Import Tool Server - opciones de importación](/import-tool.png)

## Mantenimiento automático

Una vez importado el servicio, **Devic se encarga del resto**.\
La plataforma transforma tu API en un conjunto de tools listas para usar y mantiene la sincronización de forma automática.\
Cada vez que actualices tu API, las tools del MCP se actualizarán también, sin necesidad de intervención manual.

<Card title="Sincronización continua" icon="arrows-rotate" horizontal>
  Solo necesitas mantener tu API. Devic se encargará de mantener actualizadas las tools y el servidor MCP de forma transparente.
</Card>
