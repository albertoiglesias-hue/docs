## Descripción general

La sección **Mis MCPs** te permite transformar tus **APIs** o **servicios internos** en servidores compatibles con el estándar **Model Context Protocol (MCP)**.\
Cada servidor MCP agrupa un conjunto de _tools_ (herramientas) que los modelos pueden invocar directamente desde tus agentes o asistentes.

En otras palabras, Devic convierte tu API en una interfaz estándar lista para ejecutar acciones reales sobre tus sistemas mediante lenguaje natural o flujos automatizados.

## Panel principal

La vista muestra todos los MCPs creados dentro de tu organización.\
Cada tarjeta representa un conector activo e incluye:

- **Nombre del MCP**
- **Descripción del servicio**
- **Indicador de estado** (verde = activo)
- **Accesos rápidos:** editar, eliminar o publicar.

En la parte superior izquierda encontrarás el botón **Add custom MCP**, que inicia el proceso de creación de un nuevo conector a partir de tu API.

<Card title="Crear un nuevo MCP" icon="plus" horizontal>
  Haz clic en <strong>Add custom MCP</strong>
   para comenzar la configuración de un servidor MCP basado en la especificación de tu API (OpenAPI o JSON).
</Card>

---

## Tipos de MCP en la vista

- **Mis MCPs:** conectores creados por ti, a partir de tus propios servicios o endpoints.
- **Importados:** integraciones externas cargadas desde un archivo o una URL pública, sin modificar su estructura original.

Ambos tipos se muestran organizados dentro de la misma vista, en secciones diferenciadas.
